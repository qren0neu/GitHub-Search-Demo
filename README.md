# GitHub-Search-Demo

This is a demo Android App using Jetpack Compose, Fragment, ViewModel, Coil and Retrofit.

Implementing the feature of searching user profile by username, view their followings / followers.

How to run?

in `GitHubApi.kt`, modify `API_CLIENT_SECRET` using your GitHub personal access token (PAT).
